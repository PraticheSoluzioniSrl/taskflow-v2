# TaskFlow V2

Task Manager con sincronizzazione database automatica.

## Setup

1. Copia .env.example in .env.local
2. Configura le variabili (DATABASE_URL, GOOGLE_CLIENT_ID, etc)
3. npm install
4. npm run dev

## Deploy Vercel

Configura le stesse variabili su Vercel + NEXTAUTH_URL con il dominio reale.
